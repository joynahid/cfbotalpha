Terms & Conditions

This bot is open for all. Anyone can use it for their personal usage.

This bot will store nothing but your Codeforces handle if you only perform the "Remember" query. Otherwise it won't store anything.

The bot uses Codeforces API, CLIST API and WIT API behind to serve you.

This is a open source project. Check out the source code in Github
