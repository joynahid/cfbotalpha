Welcome to CFBot Alpha

This is a bot for codeforces. Initially you can perform some useful queries in this bot. It will help you know the rating changes both predicted official based on the phase of a contest. This bot can also remember yourself if you let it to do so and perform handleless queries to make your life easier. I'll add more features real soon. So now get to the bot and perform some queries. Thanks.

What can this bot do currently?

  • ✓ Shows Contest and Handle wise rating changes both official and predicted based on the phase of that contest
  • ✓ Can remember Handle and perform queries based on that handle
  • ✓ Shows upcoming contest

Todos

  • Rating Curve of an user
  • Shows problem suggestions of an user based on his/ her rating
  • Stalk someone
